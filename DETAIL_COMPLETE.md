## 🔍 使用场景

### 场景 1：调试 API 请求
1. 客户端发起 API 请求
2. 管理端自动显示抓包
3. 点击记录查看：
   - 请求头是否正确
   - 请求参数是否完整
   - 响应状态码
   - 响应体内容（JSON 格式化）

### 场景 2：分析 HTTPS 流量
1. 客户端访问 HTTPS 网站
2. 代理自动解密（需安装 CA 证书）
3. 查看：
   - 解密后的请求数据
   - 解密后的响应数据
   - 完整 HTML/JSON 内容

### 场景 3：性能优化
1. 查看"数据大小"列
2. 识别大响应（如图片、视频、大 JSON）
3. 优化请求，减少不必要的数据传输
