# 🌐 Web 管理端使用指南

## 概述

Web 管理端提供了一个友好的图形界面，用于实时监控和查看代理抓包日志。

## 访问地址

服务器启动后，通过以下地址访问：

- **本地访问**: http://localhost:3000
- **局域网访问**: http://10.0.0.17:3000

## 功能特性

### 1. 📊 实时仪表盘

- **总请求数**: 显示抓包的总请求数量
- **HTTP 请求**: 统计 HTTP 类型请求
- **HTTPS 请求**: 统计 HTTPS 类型请求
- **错误数**: 统计响应状态码 >= 400 的请求

### 2. 📋 抓包列表

- 按时间倒序显示所有抓包记录
- 显示请求类型（HTTP/HTTPS）
- 显示请求方法（GET/POST/PUT/DELETE）
- 显示请求 URL
- 显示响应状态码（颜色标记）
- 显示请求时间

### 3. 🔍 过滤功能

- 支持按 URL、方法、类型过滤
- 实时搜索，输入即过滤

### 4. 📖 详情查看

点击任意抓包记录，展开查看详细信息：

- **请求头**: 完整的 HTTP 请求头
- **响应头**: 完整的 HTTP 响应头
- **响应体**: 响应正文内容（格式化显示）
- **HTTPS 数据**: 解密后的客户端和服务器数据

### 5. 🔄 自动刷新

- 默认每 2 秒自动刷新
- 可手动关闭自动刷新
- 点击"刷新"按钮手动刷新

### 6. 🗑️ 日志管理

- 一键清空所有抓包日志
- 清空操作有确认提示

## 使用步骤

### 1. 启动服务

**启动代理服务：**
```bash
cd /home/ubuntu/.openclaw/workspace/proxy-server
./manage.sh start
```

**启动 Web 管理端：**
```bash
./manage.sh web-start
```

**或同时启动：**
```bash
./manage.sh start
./manage.sh web-start
```

### 2. 访问管理端

在浏览器中打开：http://10.0.0.17:3000

### 3. 配置客户端

按照 `CLIENT_SETUP.md` 配置客户端代理和 CA 证书。

### 4. 开始抓包

- 客户端发起 HTTP/HTTPS 请求
- 管理端会自动显示抓包数据
- 点击记录查看详细信息

## 管理命令

```bash
# 查看状态（包含 Web 服务）
./manage.sh status

# 启动/停止/重启 Web 服务
./manage.sh web-start
./manage.sh web-stop
./manage.sh web-restart

# 查看 Web 服务日志
tail -f /tmp/web-server.log
```

## API 接口

Web 管理端提供以下 API：

### 获取日志

```
GET /api/logs
```

返回：
```json
{
  "logs": [
    {
      "type": "http",
      "method": "GET",
      "url": "http://example.com/",
      "requestHeaders": {...},
      "responseStatus": 200,
      "responseHeaders": {...},
      "responseBody": "...",
      "timestamp": "2026-07-04T12:08:34.947Z"
    }
  ]
}
```

### 清空日志

```
POST /api/logs/clear
```

返回：
```json
{
  "success": true
}
```

### 获取统计

```
GET /api/stats
```

返回：
```json
{
  "total": 100,
  "http": 60,
  "https": 40,
  "errors": 5
}
```

## 常见问题

### 无法访问管理端

1. 检查 Web 服务是否启动：`./manage.sh status`
2. 检查端口 3000 是否被占用：`netstat -tlnp | grep 3000`
3. 检查防火墙：`sudo ufw allow 3000/tcp`

### 日志不显示

1. 确认代理服务正在运行
2. 确认客户端已配置代理
3. 确认客户端已安装 CA 证书（HTTPS）
4. 点击"刷新"按钮手动刷新

### 自动刷新太频繁

取消勾选"自动刷新"复选框，手动点击"刷新"按钮。

## 安全建议

1. **访问控制**: Web 管理端默认监听所有网卡（0.0.0.0），建议仅内网访问
2. **认证**: 当前版本无认证，建议在内网使用或添加认证
3. **数据敏感**: 抓包可能包含敏感信息，请妥善保管日志
