## 🌍 公网访问配置

### 需要开放的端口（共 3 个）

| 端口 | 用途 | 协议 | 状态 |
|------|------|------|------|
| **3000** | Web 管理端 | TCP | ✅ 监听 0.0.0.0 |
| **8080** | 代理服务 | TCP | ✅ 监听 0.0.0.0 |
| **8888** | 证书下载 | TCP | ✅ 监听 0.0.0.0 |

### 配置步骤

1. **云服务器安全组**（必须）
   - 腾讯云: https://console.cloud.tencent.com/cvm/securitygroup
   - 阿里云: https://ecs.console.aliyun.com/
   - 添加入站规则，开放上述 3 个端口

2. **系统防火墙**（可选）
   ```bash
   sudo ufw allow 3000/tcp
   sudo ufw allow 8080/tcp
   sudo ufw allow 8888/tcp
   ```

3. **验证配置**
   ```bash
   # 浏览器访问
   http://115.159.196.184:3000
   
   # 或命令行测试
   curl http://115.159.196.184:3000/
   ```
