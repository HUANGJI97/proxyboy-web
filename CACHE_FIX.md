## 🎯 解决方案（按优先级排序）

### 方案 1：清除浏览器缓存（最有效）✨

#### Windows / Linux
1. 按 **`Ctrl + Shift + Delete`**
2. 选择时间范围："所有时间"
3. 勾选："缓存的图片和文件"
4. 点击"清除数据"
5. 然后按 **`Ctrl + F5`** 强制刷新

#### Mac
1. 按 **`Cmd + Shift + Delete`**
2. 选择时间范围："所有时间"
3. 勾选："缓存的图片和文件"
4. 点击"清除数据"
5. 然后按 **`Cmd + Shift + R`** 强制刷新

---

### 方案 2：使用隐私/无痕模式（推荐）✨

#### Chrome
1. 按 **`Ctrl + Shift + N`**（Windows/Linux）
2. 或 **`Cmd + Shift + N`**（Mac）
3. 在隐私窗口中访问：
   ```
   http://115.159.196.184:3000/proxyman-style.html
   ```

#### Firefox
1. 按 **`Ctrl + Shift + P`**（Windows/Linux）
2. 或 **`Cmd + Shift + P`**（Mac）
3. 在隐私窗口中访问上面的地址

**隐私模式不会加载缓存，所以一定能看到最新界面！**

### 方案 3：强制刷新（带缓存清除）

1. 按 **`F12`** 打开开发者工具
2. 在地址栏输入：
   ```
   http://115.159.196.184:3000/proxyman-style.html
   ```
3. 在页面上 **右键点击刷新按钮**
4. 选择 **"清空缓存并硬性重新加载"**

---

### 方案 4：添加随机参数（绕过缓存）

在 URL 后面添加 `?t=随机数`：

```
http://115.159.196.184:3000/proxyman-style.html?t=123456
```

或者点击这个链接：
```html
<a href="http://115.159.196.184:3000/proxyman-style.html?t=123456">访问 Proxyman 界面（绕过缓存）</a>
```

每次修改 `t=` 后面的数字，浏览器会认为是新页面。
