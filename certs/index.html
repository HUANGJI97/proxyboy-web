📖 安装步骤

📱 iOS (iPhone/iPad) 详细步骤

  1. 下载证书：点击上方按钮下载 ca.crt
  2. 发送证书到 iOS 设备：
    方法1：通过 AirDrop 发送到 iPhone/iPad
    方法2：通过 邮件 发送附件
    方法3：访问 http://115.159.196.184:8888 直接下载
  3. 安装描述文件：
    • 打开 设置 → 通用
    • 找到 "VPN与设备管理" 或 "描述文件"
    • 点击 ProxySniffer CA
    • 点击右上角 "安装"
    • 输入 设备密码
    • 点击 "安装" 确认
  4. ⚠️ 启用完全信任（重要！）：
    • 打开 设置 → 通用
    • 滑到最下方，点击 "关于本机"
    • 找到 "证书信任设置" 并点击
    • 找到 ProxySniffer CA
    • 打开右侧开关
    • 弹出警告，点击 "继续"
  5. 配置代理：
    • 打开 设置 → Wi-Fi
    • 点击当前连接的 Wi-Fi 右侧的 ⓘ 图标
    • 滑到最下方，点击 "配置代理"
    • 选择 "手动"
    • 服务器：115.159.196.184
    • 端口：8080
    • 点击右上角 "存储"
  6. 测试：打开 Safari，访问 https://example.com，如果能正常访问说明配置成功
⚠️ 常见问题：

Q1: 找不到"描述文件"？
A: iOS 16+ 路径：设置 → 通用 → VPN与设备管理

Q2: 安装后还是无法解密 HTTPS？
A: 必须完成第4步"启用完全信任"！这是最容易遗漏的步骤。

Q3: 启用完全信任时提示"无法验证"？
A: 这是正常的，因为是自签名证书。点击"继续"即可。

Q4: Safari 显示"无法连接"？
A: 检查代理地址和端口是否正确，确保代理服务正在运行。

Q5: 如何卸载证书？
A: 设置 → 通用 → VPN与设备管理 → 点击证书 → 删除描述文件

🖥️ Windows

  1. 点击下载的 ca.crt 文件
  2. 点击 "安装证书"
  3. 选择 "当前用户" 或 "本地计算机"
  4. 选择 "将所有证书放入下列存储"
  5. 点击 "浏览"，选择 "受信任的根证书颁发机构"
  6. 下一步 → 完成 → 弹出警告点击 "是"

🍎 macOS

  1. 双击 ca.crt 文件，会自动打开"钥匙串访问"
  2. 选择 "系统" 或 "登录" 钥匙串
  3. 点击 "添加"
  4. 打开 "钥匙串访问" 应用
  5. 左侧选择钥匙串 → 找到 ProxySniffer CA
  6. 双击证书 → 展开 "信任" → 选择 "始终信任"

🤖 Android

  1. 设置 → 安全 → 加密与凭据 → "安装证书" → CA 证书
  2. 选择下载的 ca.crt
  3. 命名：ProxySniffer CA → 确定
  4. 设置 → Wi-Fi → 长按当前 Wi-Fi → 修改网络
  5. 高级选项 → 代理 → "手动"
  6. 代理服务器：115.159.196.184 端口：8080
