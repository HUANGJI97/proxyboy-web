## 页面功能

1. **查看证书信息** - 显示证书详细属性
2. **一键下载** - 点击按钮下载 `ca.crt`
3. **安装指引** - 提供各平台安装步骤
4. **安全提醒** - 提示安全注意事项
5. **返回链接** - 快速返回 Web 管理端

## 技术实现

- **纯 HTML + CSS** - 无需 JavaScript
- **Node.js 服务器** - `cert-server.js`
- **自动 MIME 类型** - 正确识别 `.crt` 文件
- **附件下载** - 设置 `Content-Disposition` 头
