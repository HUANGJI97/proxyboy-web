# 🎨 Proxyman 风格界面 - 功能说明

## 界面预览

新界面采用 **深色主题**，模仿 Proxyman 的专业风格，提供更好的视觉体验和功能性。

---

## ✨ 核心功能

### 1. 左侧请求列表

**显示内容：**
- ✅ HTTP 方法标签（GET/POST/PUT/DELETE/HTTPS）
- ✅ 请求 URL（自动截断超长 URL）
- ✅ 状态码（200/404/500 等，颜色区分）
- ✅ 时间戳
- ✅ 数据大小

**交互功能：**
- 👆 **点击选择**：点击任意请求，右侧显示详情
- 🎨 **高亮选中**：当前选中的请求会高亮显示
- 🔍 **实时过滤**：顶部搜索框支持按 URL、方法、状态码过滤

---

### 2. 右侧详情面板

#### 标签 1：Overview（概览）
```
┌─────────────────────────────────┐
│  General                       │
│  - Request URL                │
│  - Method                     │
│  - Status Code               │
│  - Protocol                  │
├─────────────────────────────────┤
│  Request Headers             │
│  (表格形式展示)              │
├─────────────────────────────────┤
│  Query Parameters            │
│  (自动解析 URL 参数)         │
├─────────────────────────────────┤
│  Request Body               │
│  (JSON 语法高亮)            │
└─────────────────────────────────┘
```

#### 标签 2：Request（请求详情）
- ✅ 请求头表格
- ✅ 请求体（JSON 格式化 + 语法高亮）
- ✅ 原始客户端数据

#### 标签 3：Response（响应详情）
- ✅ 响应头表格
- ✅ 响应体（JSON 语法高亮）
- ✅ 原始服务器数据

#### 标签 4：Raw Data（原始数据）
- ✅ 完整的 JSON 日志
- ✅ 一键复制按钮

---

### 3. 顶部工具栏

#### 🔍 搜索框
- **功能**：实时过滤请求列表
- **支持过滤**：
  - URL 关键词
  - HTTP 方法（GET/POST 等）
  - 请求类型（http/https）
- **示例**：
  - 输入 `api` → 显示包含 "api" 的请求
  - 输入 `POST` → 显示所有 POST 请求
  - 输入 `200` → 显示状态码为 200 的请求

#### 📋 Copy as cURL 按钮
- **功能**：将选中的请求导出为 cURL 命令
- **复制内容**：
  ```bash
  curl -X POST \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer xxx" \
    -d '{"key": "value"}' \
    "https://api.example.com/endpoint"
  ```
- **使用场景**：
  - 快速复现请求
  - 分享给同事
  - 用于 Postman/Insomnia 导入

#### 🗑️ Clear 按钮
- **功能**：清空所有抓包日志
- **确认提示**：点击后会弹出确认对话框

## 🎨 界面特点

### 深色主题
- **背景色**：`#1e1e1e`（VS Code 深色风格）
- **侧边栏**：`#252526`
- **高亮色**：`#569cd6`（蓝色，用于链接和选中状态）
- **成功色**：`#4ec9b0`（绿色，用于 2xx 状态码）
- **错误色**：`#f44747`（红色，用于 4xx/5xx 状态码）

### 专业配色
| 元素 | 颜色 | 用途 |
|------|------|------|
| HTTP 方法 | 彩色标签 | GET=蓝, POST=绿, PUT=橙, DELETE=红 |
| 状态码 | 绿色/红色 | 2xx=绿, 4xx/5xx=红 |
| JSON 键 | `#9cdcfe` | 浅蓝色 |
| JSON 字符串 | `#ce9178` | 橙色 |
| JSON 数字 | `#b5cea8` | 浅绿色 |

## 🚀 使用指南

### 快速开始

1. **访问界面**：
   ```
   http://115.159.196.184:3000/proxyman-style.html
   ```

2. **开始抓包**：
   - 配置设备代理为 `115.159.196.184:8080`
   - 安装 CA 证书（如需抓取 HTTPS）

3. **查看请求**：
   - 左侧列表会实时显示所有请求
   - 点击任意请求查看详情

---

### 常见操作

#### 过滤特定请求
1. 在搜索框输入关键词
2. 列表实时过滤
3. 示例：
   - `google.com` → 显示包含 "google.com" 的请求
   - `POST` → 显示所有 POST 请求
   - `404` → 显示所有 404 响应

#### 查看 JSON 数据
1. 点击请求
2. 切换到 **Overview** 或 **Response** 标签
3. JSON 会自动格式化并语法高亮

#### 导出为 cURL
1. 选中请求
2. 点击顶部 **📋 Copy as cURL** 按钮
3. cURL 命令自动复制到剪贴板
4. 粘贴到终端测试

#### 复制原始数据
1. 切换到 **Raw Data** 标签
2. 点击 **Copy** 按钮
3. 完整的 JSON 日志被复制
